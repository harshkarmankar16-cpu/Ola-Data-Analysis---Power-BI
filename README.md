# Ola-Data-Analysis---Power-BI
This project analyzes Ola ride-booking data to uncover business insights related to ride trends, customer behavior, cancellations, revenue, and driver performance. The dataset was cleaned and transformed using SQL, then visualized in Power BI to create an interactive dashboard that supports data-driven decision-making.
